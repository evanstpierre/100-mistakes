## Chapter 1 - Go Simple to learn hard to master

**7 main mistakes:**
* Bugs
* Needless Complexity
* Weaker readability 
* Suboptimal or unidomatic organization
* Lack of API convenience
* Under-optimized code
* Lack of Productivity
